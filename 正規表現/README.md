# 正規表現について
正規表現とは、文字列内で文字の組み合わせを照合するために用いられるパターンです。
vscodeで正規表現で検索ができ、好みの組み合わせで検索が行える。


## 正規表現で使用する文字

```
.   ^   $   [   ]   *   +   ?   |   (   )
```
↑をメタ言語という、

- `.` → なんでもいい一文字
- `^` → 文字列の先頭を指す
- `$` → 文字列の先頭を指す
